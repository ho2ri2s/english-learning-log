# English Review for Today

今日の英語学習記録を添削します。

## 実行手順

1. 今日の日付のファイル（`daily/YYYY-MM-DD.md`）を読み込む
2. 英作文セクションを分析し、以下を確認:
   - 文法エラー
   - スペルミス
   - より自然な言い回し
   - 語彙の適切性
3. 添削結果を `reviews/YYYY-MM-DD-review.md` に保存
4. CLI上で以下のフォーマットで結果を表示:

```
## 🏁 添削結果
[全体的な評価]

## 📝 修正箇所リスト
- Line X: 修正内容
- Line Y: 修正内容

## 💻 添削後の英文
[修正後の完全な英文]

## 📖 修正理由の詳細
1. 各修正点の詳細な説明
2. 文法ルールや慣用表現の解説

## 💡 追加のアドバイス
[特に添削して欲しいポイントへの回答や、表現のバリエーション]
```

## 出力ファイルフォーマット

`reviews/YYYY-MM-DD-review.md` には以下の形式で保存:

```markdown
# YYYY-MM-DD Review

## 元の英作文

[元の文章]

## 添削

### 文法・スペルの修正

- ❌ Line X: [誤り]
- ✅ 修正: [正しい表現]
- 📝 理由: [説明]

### より自然な表現

[改善提案]

## 添削後の完成版

[修正後の完全な英文]

## シャドーイングで使えるフレーズ

### 今日学んだフレーズ
- [フレーズ1]: 意味と使い方
- [フレーズ2]: 意味と使い方

### 発音のポイント
- [注意すべき発音]

## その他のフィードバック

[全体的なアドバイスや励まし]
```

## 注意事項

- 日本語で出力すること
- 具体的で建設的なフィードバックを提供
- 修正理由を明確に説明
- ユーザーの「特に添削して欲しいポイント」に必ず回答
