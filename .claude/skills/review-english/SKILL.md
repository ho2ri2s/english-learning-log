# English Review Skill

このスキルは、英語学習の日次記録を添削し、改善提案を行います。

## 使い方

dailyディレクトリのマークダウンファイルを読み取り、以下を行います:
1. 英作文セクションの添削
2. より自然な表現の提案
3. シャドーイングで使えるフレーズの抽出
4. reviewsディレクトリに結果を保存

## 出力フォーマット
```markdown
# YYYY-MM-DD Review

## 英作文の添削

### 元の文
...

### 添削
- ❌ 誤り: ...
- ✅ 改善: ...

### より良い表現例
...

## シャドーイングフレーズ

### 使えるフレーズ
- ...

### 発音のポイント
- ...

## その他のフィードバック

...
```

## 実行手順

1. dailyファイルを確認
2. 以下を分析:
   - 文法エラー
   - より自然な言い回し
   - 語彙の適切性
3. reviewsディレクトリに保存
